T1048: Control plane signaling exfiltration
Architecture Segment:  Control-plane, Roaming
----------------------------------------
Procedure Examples
----------------------------------------
HTTP/2 optional parameters: HTTP/2 optional parameters may be used to communicate between a core NF and an external server (AF, IPX node) via NEF or SEPP.
----------------------------------------
Mitigations
----------------------------------------
M1037: Filter and inspect network traffic coming out of SEPP and NEF
----------------------------------------

----------------------------------------
----------------------------------------
Critical Assets

Operator resource identifiers: Core NF identifiers (e.g. IP address, FQDN) such as AMF, SMF, UDM etc.

DS0029: Monitor and analyze traffic patterns and packet inspection over the SBI, especially to/from external functions.
