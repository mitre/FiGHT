T1608.501: Configurability of Fake Base Station or Access Point
Architecture Segment:  RAN
----------------------------------------
Procedure Examples
----------------------------------------
Obtain capability for configuration of gNB, gNB emulator, or WiFi access point.: Adversary obtains software capability such as: modified custom code, scripts, configuration parameters.
Configure fake gNB to appear strongest: Adversary installs an illegitimate complete gNB and configures the power so as to appear strongest to a given UE in a particular location, see [2]
