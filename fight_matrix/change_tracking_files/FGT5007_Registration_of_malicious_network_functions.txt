FGT5007: Registration of malicious network functions
Architecture Segment:   OA&M, Control plane
----------------------------------------
Procedure Examples
----------------------------------------
Unauthorized use of API: Rogue NF calls Nnrf_NFManagement API to register one of these functions: AMF, SMF, UDM, AUSF, NEF, PCF, SMSF, NSSF, UPF, etc. Clause 5.2.7 of [3]
----------------------------------------
Mitigations
----------------------------------------
FGM1506: Cross check newly registered NFs. Out of band mechanism for cross checking new NFs that are registered in NRF are as expected by network administrator.
M1018: User account access
M1030: Network segmentation
----------------------------------------
Pre-Conditions
----------------------------------------
----------------------------------------
Critical Assets
----------------------------------------
Network services: Network services provided to UEs.
----------------------------------------
Detection
----------------------------------------
DS0015: Detections for T1610. Monitor application logs of core NFs.
DS0032: Detections for T1610--Deploy Container. Applicable only in the case that the NF is deployed as a container.
